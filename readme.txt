== Description ==
**BigPurple AI Writer** is a simple WordPress plugin that integrates the [OpenAI](https://openai.com/) GPT-3.5 Turbo model into your WordPress admin panel. With this plugin, you can interact with ChatGPT, the AI language model, directly within your site's dashboard. It eliminates the need to switch between tabs, making content creation a seamless and efficient process.

[OpenAI TOS](https://openai.com/policies/terms-of-use)
[OpenAI Privacy Policy](https://openai.com/policies/privacy-policy)

Features:

- Integrate OpenAI's GPT-3.5 Turbo model into your WordPress admin panel.
- Communicate with ChatGPT through a convenient link in the top bar.
- Get real-time responses for your queries, from creative writing to research insights.
- Copy and paste the generated responses directly into your content.
- Easy-to-use interface for a smooth user experience.

**IMPORTANT**: Before using this plugin, you need to have an OpenAI account and an API key. If you don't have one, sign up for an account at [OpenAI Signup](https://platform.openai.com/signup) and obtain your API key from [OpenAI API Keys](https://platform.openai.com/account/api-keys).

Once you have your OpenAI API key, follow these steps to set up the plugin:

1. Install and activate the "BigPurple AI Writer" plugin from the WordPress plugin directory.
2. Go to **Settings > BigPurple AI Writer** in your WordPress admin panel.
3. Enter your OpenAI API key in the field labeled **OpenAI API Key**.
4. Save the settings.

You are now ready to use BigPurple AI Writer and benefit from its powerful AI capabilities for content creation!

== Installation ==

After activating the plugin, go to **Settings > BigPurple AI Writer** in your WordPress admin panel. Then, enter your OpenAI API key in the field labeled **OpenAI API Key**.
